# Long-Term Vision


## Features

### Context gathering, storage and retrieval

- Understand WhatsApp messages (can be extended to voice, video calls and IRL conversations).
- Store this context in a manner such that it's secure and has well-defined retrieval patterns.

### Act as a second brain, via AI

- Uses common AI patterns like agents to support any potential requirement that can be fulfilled using the
  above-mentioned information.

## Example use cases

- Remember and remind a user about commitments made
- Summarise conversations
- Bring context from one conversation to another
- Take action for the user (Ex: Mom asked me to buy groceries from blinkit)
